# Dieto Metrics v2.0 Calculadora nutricional con base de datos de 2,863 alimentos mexicanos (INCMNSZ). ## Estructura ``` dieto_metrics/ ├── backend/ │ ├── main.py ← FastAPI — API REST │ ├── requirements.txt │ └── alimentos5.db ← Base de datos SQLite (copiar aquí) ├── frontend/ │ ├── src/ │ │ ├── App.jsx │ │ ├── api.js │ │ └── components/ │ │ ├── Sidebar.jsx │ │ ├── FoodTable.jsx │ │ ├── NutritionSummary.jsx │ │ ├── MacroChart.jsx │ │ └── Toast.jsx │ └── package.json └── start.sh ← Arrancar todo con un comando ``` ## Requisitos - Python 3.9+ - Node.js 18+ - pip ## Arrancar (opción fácil) ```bash chmod +x start.sh ./start.sh ``` Luego abre: **http://localhost:5173** ## Arrancar manualmente **Backend:** ```bash cd backend pip install -r requirements.txt uvicorn main:app --reload --port 8000 ``` **Frontend (en otra terminal):** ```bash cd frontend npm install npm run dev ``` ## API endpoints | Método | Ruta | Descripción | |--------|------|-------------| | GET | `/grupos` | Lista de grupos de alimentos | | GET | `/alimentos?grupo=X` | Alimentos por grupo | | GET | `/alimento/{nombre}` | Detalle de un alimento | | POST | `/calcular` | Calcular macros de una lista | | GET | `/docs` | Documentación interactiva (Swagger) | ## Fase siguiente - [ ] Autenticación JWT (usuario / admin) - [ ] Sistema de licencias por device hash - [ ] Panel de administración - [ ] Empaquetado con Electron (desktop)
